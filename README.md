# Onco_education — ФОС кафедры онкологии (31.08.57)

**Сайт (GitHub Pages):** https://evdokimov-onco1.github.io/Onco_education/ — развёрнут, дашборд подгружает `fos-data.json`.

Рабочая среда для подготовки фондов оценочных средств (ФОС) к аккредитации. Один репозиторий связывает три вещи: **сайт-дашборд**, **общие данные** и **документы в Google Drive**.

## Как устроено

```
GitHub (этот репозиторий)  ← единый источник правды
├── index.html        сайт-дашборд (GitHub Pages)
├── fos-data.json     распределение тем, ответственные, приёмка, ссылки на Drive
├── README.md         этот файл
└── DEVIN.md          задание для агента Devin

        index.html читает fos-data.json и даёт ссылки в ▼
Google Drive: папка «ФОС Онкология 31.08.57»
└── 8 Google-документов — здесь преподаватели пишут содержание ФОС
        https://drive.google.com/drive/folders/1NMOVu7uhu1TAbN3jm3YKBlswrtEaaG-A

Devin — агент, который правит репозиторий (деплой, обновление данных, генерация заготовок)
```

**Разделение ролей.** Текст ФОС (тесты, вопросы, задачи) пишут в Google-документах. Кто за какую тему отвечает и в каком статусе — в дашборде. `fos-data.json` — общий файл, который синхронизирует картинку для всех.

## Почему без Notion и без сервера

Общее состояние хранится в `fos-data.json` прямо в репозитории — «git как база данных». Все видят одинаковые данные, потому что сайт грузит этот файл. Не нужны ни Notion (который у вас требует VPN), ни отдельный backend, ни база данных.

## Как обновлять распределение

1. Откройте сайт, поменяйте ответственных / отметьте приёмку.
2. Нажмите **«Экспорт назначений»** — скачается `fos-data.json`.
3. Замените им файл в репозитории (через сайт GitHub: открыть файл → карандаш → вставить → Commit) — **или** поручите это Devin.
4. Через минуту сайт покажет обновлённые данные.

## Как включить сайт (GitHub Pages)

1. **Settings → Pages → Source: Deploy from a branch → `main` → `/(root)` → Save.**
2. Через минуту сайт будет по адресу `https://evdokimov-onco1.github.io/Onco_education/`.

## Файлы для первого коммита

Положите в корень репозитория: `index.html`, `fos-data.json`, `README.md`, `DEVIN.md`.
