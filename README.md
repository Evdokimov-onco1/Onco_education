# Onco_education — ФОС кафедры онкологии (31.08.57)

Среда для распределения работы над фондами оценочных средств между врачами кафедры.

**Сайт:** `https://evdokimov-onco1.github.io/Onco_education/` (после включения GitHub Pages)
**Документы ФОС:** [папка в Google Drive](https://drive.google.com/drive/folders/1NMOVu7uhu1TAbN3jm3YKBlswrtEaaG-A)

## Что решает

Один ФОС — это 80 тестов, 30 вопросов и 15 задач по 5–7 темам. Раньше было непонятно, кто за что отвечает *внутри* документа. Теперь каждая тема закреплена за врачом **с точными номерами заданий**: например, Ронзин в ФОС 1 семестра ведёт тему «Рак желудка и пищевода» → тесты 2, 6, 10, 48, 52–80; вопросы 20–24; задачи 1, 6.

Номера получены дословным разбором файлов ФОС, а не назначены произвольно.

## Структура

```
index.html      сайт-дашборд, читает fos-data.json
fos-data.json   единый источник правды: темы → номера заданий → ответственный → ссылка на Drive
fos/            реальные тексты пяти существующих ФОС
playbooks/      generate-fos.md (спец формата) · knowledge.md (контекст для Devin)
DEVIN.md        задачи для Devin
.github/        workflow запуска Devin через API
```

## Как работает связка

**GitHub** хранит данные и сайт → **сайт** показывает, кто над чем работает, и ведёт в → **Google Drive**, где врачи пишут содержание. **Devin** обслуживает репозиторий: деплой, обновление данных, черновики недостающих ФОС.

Общее состояние лежит в `fos-data.json` — «git как база данных». Ни сервера, ни Notion, ни VPN.

## Запуск

1. **Pages:** Settings → Pages → Deploy from a branch → `main` → `/(root)`.
2. **Devin:** Settings → Secrets and variables → Actions → секрет `DEVIN_API_KEY`. Затем Actions → «Devin — запуск задачи» → Run workflow.
3. **Drive:** открыть доступ врачам на папку; Devin (задача 2 в DEVIN.md) зальёт реальные ФОС и пропишет ссылки.

## Известные дефекты исходных ФОС

Обнаружены при разборе, кафедрой приняты к сведению:
- **Сквозные задания** — часть вопросов (реабилитация, организация) не относится ни к одной теме своего ФОС. В 3 семестре таких 32 из 80. На сайте помечены пунктиром.
- **Перекос покрытия** — в 1 семестре на ЖКТ приходится ~60% заданий, на организацию онкопомощи — 2 задания из 80.
